# Journal

A minimalistic command-line interface (CLI) for journaling with Markdown.

## Installation

```
git clone https://github.com/christianbundy/journal.git
```

## Usage

``` shell
bin/journal
```

``` shell
bin/journal yesterday
```

``` shell
bin/journal october 21 2015
```
